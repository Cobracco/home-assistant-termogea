# Support

## Per utenti

Per problemi d’uso o richieste funzionali:

- apri una issue GitHub
- includi:
  - dominio coinvolto (`termogea`, `voice_orchestrator`, `aruba-ddns`)
  - versione Home Assistant
  - log rilevanti
  - passi per riprodurre il problema

## Per bug di integrazione

Allega quando possibile:

- configurazione rilevante senza segreti
- stack trace
- screenshot UI
- versione del repository o commit
